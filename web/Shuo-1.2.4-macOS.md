# Shuo 1.2.4 Release Notes

## English

Adds Groq, SiliconFlow, and custom OpenAI-compatible endpoint support, with improved cloud model and API key configuration.

## 简体中文

新增 Groq、硅基流动和自定义 OpenAI-compatible 端点支持，并完善云端模型与 API 密钥配置。

## 繁體中文

新增 Groq、矽基流動與自訂 OpenAI 相容端點支援，並完善雲端模型與 API 金鑰設定。

## 日本語

Groq、SiliconFlow、カスタム OpenAI 互換エンドポイントに対応し、クラウドモデルと API キーの設定を改善しました。
