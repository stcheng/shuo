# Shuo 1.3.0 Release Notes

## English

- Improves automatic switching between cloud services.
- Reduces false transcripts from recordings without speech.
- Adds a System language option for new Shuo installations.
- Improves the stability of live audio capture from USB microphones.

## 简体中文

- 优化云端服务之间的自动切换体验。
- 减少空录音被误转写为文字的情况。
- 应用语言现已支持“跟随系统”选项。
- 提升 USB 麦克风实时音频采集稳定性。

## 繁體中文

- 優化雲端服務之間的自動切換體驗。
- 減少空錄音被誤轉寫為文字的情況。
- App 語言現已支援「跟隨系統」選項。
- 提升 USB 麥克風即時音訊擷取穩定性。

## 日本語

- クラウドサービス間の自動切り替えをよりスムーズにしました。
- 無音または空の録音が誤って文字起こしされる問題を減らしました。
- アプリの言語に「システム設定」の選択肢を追加しました。
- USB マイクからのライブ音声キャプチャの安定性を改善しました。
