# Shuo 1.2.2 Release Notes

## English

- Added a Settings toggle for showing Shuo in the Dock while keeping the default menu-bar-only behavior.
- Check for Updates now shows “You’re up to date” inline in Settings when no update is available.
- Preserved model-provided sentence-ending punctuation in mixed Chinese/English transcripts.
- Kept the compact Floating Bar cursor as an arrow over the logo.

## 简体中文

- 新增是否在 Dock 中显示 Shuo 的设置开关，同时默认保持仅菜单栏显示。
- 没有可用更新时，“检查更新”会直接在设置页面显示“已是最新版本”。
- 中英文混合转写会保留模型已经给出的句末标点。
- 悬浮栏只显示 logo 时，鼠标悬停会保持箭头光标。

## 繁體中文

- 新增是否在 Dock 中顯示 Shuo 的設定開關，同時預設維持僅顯示於選單列。
- 沒有可用更新時，「檢查更新」會直接在設定頁面顯示「已是最新版本」。
- 中英文混合轉寫會保留模型已給出的句末標點。
- 懸浮列只顯示 logo 時，滑鼠懸停會維持箭頭游標。

## 日本語

- 初期設定はメニューバーのみのまま、Dock に Shuo を表示する設定を追加しました。
- 利用可能なアップデートがない場合、「アップデートを確認」は設定画面内に最新状態を表示します。
- 中国語と英語が混在する文字起こしで、モデルが出力した文末句読点を保持します。
- コンパクトなフローティングバーでは、ロゴ上のカーソルを矢印のまま維持します。
