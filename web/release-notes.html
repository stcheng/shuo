更清晰的本地模型选择更清楚的本機模型選擇ローカルモデルを、より選びやすくA clearer local-model choice

  • 新增 SenseVoice Small：适合中文、日文与混合语音的完全本地转写选项，但不支持上下文、词库和项目术语提示。新增 SenseVoice Small：適合中文、日文與混合語音的完全本機轉寫選項，但不支援上下文、詞庫與專案術語提示。中国語、日本語、混在音声向けの完全ローカルモデル SenseVoice Small を追加しました。文脈、用語集、プロジェクト用語ヒントには対応していません。Added SenseVoice Small, a fully local option for Chinese, Japanese, and mixed speech. It does not support prompt context, custom vocabulary, or project-terminology hints.
  • 本地下载模型现在收敛为 SenseVoice Small、Whisper Small 与 Whisper Large Turbo 三种清晰选择。Shuo 会根据所选语言与 Mac 配置给出推荐。本機下載模型現在收斂為 SenseVoice Small、Whisper Small 與 Whisper Large Turbo 三種清楚的選擇。Shuo 會依所選語言與 Mac 配置給出建議。ダウンロードできるローカルモデルは、SenseVoice Small、Whisper Small、Whisper Large Turbo の 3 つに整理しました。選択した言語と Mac に応じて Shuo が候補を提案します。Local downloads are now intentionally limited to SenseVoice Small, Whisper Small, and Whisper Large Turbo. Shuo recommends one based on your selected languages and Mac.
  • 下载前即可看到每个模型的语言范围、体积，以及是否支持上下文提示和自定义词汇。下載前即可查看每個模型的語言範圍、大小，以及是否支援上下文提示與自訂詞彙。ダウンロード前に、各モデルの対応言語、サイズ、文脈ヒント・カスタム用語への対応を確認できます。Before downloading, each model explains its language range, size, and support for context hints and custom vocabulary.

更可靠的输入与粘贴更可靠的輸入與貼上より安定した入力と貼り付けMore dependable input and paste

  • “系统默认”现在是默认音频输入选择，让外接麦克风和 AirPods 更稳定地跟随 macOS 路由。「系統預設」現在是預設音訊輸入選擇，讓外接麥克風和 AirPods 更穩定地跟隨 macOS 路由。音声入力の初期値を「システムのデフォルト」に変更しました。外部マイクや AirPods が macOS の入力経路により安定して追従します。System Default is now the default audio-input choice, making external microphones and AirPods follow macOS routing more predictably.
  • 改进了麦克风启动与音频格式变化的处理，降低蓝牙或外接麦克风丢失开头语音的概率。改進了麥克風啟動與音訊格式變化的處理，降低藍牙或外接麥克風遺失開頭語音的機率。マイク起動時と音声フォーマット変更時の処理を改善し、Bluetooth や外部マイクで発話の冒頭が欠ける可能性を減らしました。Improved microphone startup and audio-format changes so a Bluetooth or external microphone is less likely to lose the opening of an utterance.
