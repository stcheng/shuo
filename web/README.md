# Shuo website

This directory contains the static product site, privacy policy, release notes,
and Sparkle appcast served by GitHub Pages.

Preview the source site locally:

```sh
python3 -m http.server --directory web 8080
```

Then open http://127.0.0.1:8080.

Localized static pages are generated during the site build; edit the source
files in this directory rather than generated output.
