01 / 本地本機ローカルLocal

设置、录音、文字、历史、统计，一切都在本机；没有账号登录，没有信息收集，更没有广告。选择本地模型，无需任何网络，一样大胆开麦。 設定、錄音、歷史、統計、專案索引與修正紀錄都留在本機；無需帳號，不向 Shuo 傳送遙測或行為分析，也沒有廣告。 設定、音声、履歴、統計、プロジェクト索引、修正記録はローカルに保存。アカウント不要で、說 シュオへテレメトリや行動分析を送らず、広告もありません。 Settings, audio, history, metrics, project indexes, and corrections stay local—no account, telemetry or behavioral analytics sent to Shuo, and no ads.

02 / 云端雲端クラウドCloud

选择云端转写，只发送当前任务所需的音频、设置、启用的上下文和有限拼写提示给云端转写服务商。Shuo 听不到，看不到，碰不到。 選擇雲端轉寫時，只傳送目前任務所需的音訊、設定、已啟用的上下文與有限拼寫提示；專案原始檔與路徑不上傳。 クラウド文字起こしを選ぶと、現在の処理に必要な音声、設定、有効なコンテキスト、限定された綴りのヒントだけを送信します。ソースファイルとパスは送りません。 Cloud transcription receives the current task's audio, settings, enabled context, and bounded spelling hints—not project source or paths.

03 / 控制控制管理Control

一切设置，详尽、公开、透明。开启你需要的功能，关闭你不需要的功能。你控制一切。 API Key 存在鑰匙圈。修正學習預設關閉，模式需逐項開啟並通過安全門檻；未來訓練或上傳也需明確選擇。 API Key はキーチェーンに保存。修正学習は初期設定でオフで、各パターンは有効化と安全基準の通過が必要です。将来の学習やアップロードにも明示的な選択が必要です。 API keys live in Keychain. Correction learning is off by default; each pattern must be enabled and pass safety gates. Future training or upload also requires explicit opt-in.
